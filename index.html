Quand utiliser "Phoenix" ?

Idéal pour chasser le début d'un nouveau cycle. Les signaux sont rares (souvent 0). Si une action apparaît ici avec un volume > x2.0, c'est un signal puissant. Risque plus élevé de faux départ.

Quand utiliser "Pullback" ?

Idéal pour le "Swing Trading" tranquille. Vous achetez une tendance saine qui respire. Le taux de réussite est élevé, les gains sont modérés (5-10%). C'est la stratégie "Rente".
